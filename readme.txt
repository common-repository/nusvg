SVG image manager. PHP methods are available in order to retrieve images with parameters.

== Description ==
Nusvg plugin allows you to manage parametrable SVG files and retrieve them with parameters.

== Features ==
- Custom parameters can be passed to image (i. e. width, height, color...)
- Images can be retrieved in HTML format or as a base64 string

nusvg.pot file available

== Installation ==
1. Upload the full directory into your '/wp-content/plugins' directory
2. Activate the plugin at the plugin administration page

You will find 'Nusvg' menu in your WordPress admin panel
